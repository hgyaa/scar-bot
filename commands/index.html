Variables Reference

Variables are dynamic placeholders you can use inside bot messages, welcome messages, bump reminders, and more. Wrap them in curly braces like {user.name}. Helpers like {lower(text)} can transform values.

User Variables10 vars
{user.mention}Mentions the user. e.g. @scarbot
{user.name}The user's display name (nickname or username).
{user.username}The user's raw username (no nickname).
{user.id}The user's Discord ID.
{user.avatar}Direct URL to the user's avatar image.
{user.created_at}Date the Discord account was created.
{user.joined_at}Date the user joined the server.
{user.discriminator}Legacy discriminator (legacy accounts only).
{user.bot}Returns "true" if the user is a bot.
{user.tag}Full tag, e.g. username#0000 (legacy).
Guild / Server Variables8 vars
{guild.name}The server's name.
{guild.id}The server's Discord ID.
{guild.member_count}Total number of members.
{guild.icon}Direct URL to the server icon.
{guild.owner}Mentions the server owner.
{guild.created_at}Date the server was created.
{guild.boost_count}Number of active server boosts.
{guild.boost_level}Current boost level (0, 1, 2, or 3).
Date & Time Variables5 vars
{date}Current date, e.g. 2025-06-01
{time}Current UTC time, e.g. 14:30:00
{datetime}Date + time combined, e.g. 2025-06-01 14:30:00
{unix}Unix timestamp (seconds since epoch).
{day}Current day of week, e.g. Monday
Helpers4 helpers
{lower(text)}Converts text to lowercase. e.g. {lower({user.name})}
{upper(text)}Converts text to UPPERCASE.
{title(text)}Title Cases The Text.
{ordinal(number)}Converts a number to ordinal. e.g. 1 → 1st, 42 → 42nd
