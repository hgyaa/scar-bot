Introduction

Welcome to scar — a feature-rich Discord bot for modern servers

Welcome to scar

scar is a lightweight, powerful Discord bot built to help you run your server effortlessly. From banning rule-breakers to welcoming new members, managing your economy to setting up auto-responses — scar does it all.

By default, your server prefix is , (comma). You can change this any time with ,prefix [new prefix].

ℹ️
By default your server prefix is , — you can mention scar to see your current prefix.

Key Features

🛡️ Moderation

  • Ban, Kick, Timeout: Full moderation toolkit with DM notifications and custom messages
  • Warn System: Issue, view, remove and clear warnings with unique IDs
  • Jail System: Strip roles and confine members to a dedicated jail channel
  • Word & Link Filter: Auto-delete messages containing filtered words or Discord invites
  • Lockdowns: Lock/unlock individual channels or the entire server at once

💰 Economy

  • Coin System: Every member has a balance tracked server-wide
  • Daily Rewards: Claim $5,000 every 24 hours with ,bag daily
  • Gambling: Coin-flip with up to $250,000 stakes
  • Leaderboard: Top 10 richest members with exemption support
  • Transfers: Send coins to other users with confirmation prompts

👋 Auto Welcome

  • Welcome new members in up to 4 channels simultaneously
  • Full embed script syntax support for rich messages
  • Variable support: {user.mention}, {guild.count}, and more
  • Test your welcome message any time before it goes live

🔧 Utility & More

  • Embed Builder: Interactive button-based embed creator with live preview
  • Autoresponder: Keyword-triggered auto-replies
  • Bump Reminder: Auto-notify when it's time to bump on Disboard
  • Vanity Rewards: Auto-assign roles to members with your vanity in their bio
  • Server Logging: Full audit log to a dedicated channel

Getting Started

Getting scar running on your server takes under a minute:

  • Click Add to Discord and authorise scar with the required permissions
  • Your prefix is , by default — run ,help to see the full menu
  • Grant staff permissions with ,mod let @user or ,admin let @user
  • Browse the sidebar to configure individual features
⚠️
Heads up: scar needs Administrator or specific role permissions to run certain commands. Make sure the bot role is positioned above any roles it needs to manage.

Features › Variables

Variables

Dynamic placeholders for messages, welcomes, and more

ℹ️
Wrap variables in curly braces: {user.name}. Use helper functions like {lower({user.name})} to transform values.

User Variables

  • {user} / {user.name} — The user's username
  • {user.mention} — Pings the user
  • {user.id} — Their unique Discord ID
  • {user.display_name} — Nickname if set, else username
  • {user.avatar} — URL of their global avatar
  • {user.joined_at} — Date they joined the server
  • {user.top_role} — Their highest role name

Server Variables

  • {guild.name} — Server name
  • {guild.count} — Total member count
  • {guild.id} — Server ID
  • {guild.boost_count} — Number of active boosts
  • {guild.boost_tier} — Boost tier level

Date & Time

  • {date.now} — Current date (MM/DD/YYYY)
  • {time.now} — Current time (HH:MM AM/PM)
  • {date.utc_timestamp} — Unix timestamp in seconds

Helper Functions

  • {lower(text)} — Convert to lowercase
  • {upper(text)} — Convert to uppercase

Moderation › AntiNuke

AntiNuke

Real-time raid protection that detects and punishes attackers automatically

How It Works

AntiNuke monitors your server's audit log in real time. When someone performs a destructive action (mass bans, channel deletes, role deletes, etc.) above a configurable threshold within a 10-second window, scar automatically punishes them before they can do more damage.

🛡️
Enable it first: Run ,antinuke toggle on — nothing is active until you do.

Permission Tiers

  • Server Admin / Bot Owner — Full access. Can add/remove AN admins, reset settings, everything.
  • AntiNuke Admin — Can change all settings (modules, thresholds, punishment, log). Automatically bypass punishment. Cannot add other AN admins or run cleanup.
  • Trusted / Whitelisted — Exempt from punishment only. Zero settings access. ,trust and ,whitelist are identical aliases for this tier.

Toggle & Punishment

,antinuke toggle on/off
Enable or disable the entire antinuke system.
Admin
,antinuke punishment ban/kick/strip
ban — permanently bans the attacker. kick — kicks them. strip — removes all dangerous roles (admin, ban members, manage server, etc.).
AN Admin
,antinuke log #channel
Set a channel to receive logs whenever antinuke triggers and punishes someone.
AN Admin
,antinuke log off
Remove the log channel.
AN Admin

Modules

Each protection type is a toggleable module. All are on by default once you enable antinuke.

,antinuke <module> on/off
Toggle a single module. Modules: ban kick channel channelcreate role rolecreate webhooks emoji guild integration botadd memberprune vanity
AN Admin
,antinuke all on/off
Toggle every module on or off at once.
AN Admin

Thresholds

Thresholds control how many times an action can happen within 10 seconds before antinuke fires. Defaults are conservative — lower them for stricter protection.

,antinuke threshold
View all current thresholds. Custom values are marked with ✏️.
AN Admin
,antinuke threshold <action> <number>
Set a specific threshold. e.g. ,antinuke threshold ban 2 — punish after 2 bans in 10s. Valid actions: ban, kick, channel, channelcreate, role, rolecreate, webhooks, emoji, botadd, memberprune.
AN Admin

Default Thresholds

  • Ban / Kick / Channel Delete / Role Delete — 3 actions in 10s
  • Channel Create / Role Create / Emoji Delete — 5 actions in 10s
  • Webhooks — 3 in 10s
  • Bot Add / Member Prune — 1 (triggers instantly)

Managing Admins & Whitelist

,antinuke admin @user
Grant AntiNuke Admin access — they can change settings and are automatically whitelisted from punishment.
Admin
,antinuke unadmin @user
Remove AN Admin access (also removes their auto-whitelist).
Admin
,antinuke trust @user
Whitelist a user from punishment only — no settings access. Identical to ,antinuke whitelist @user.
AN Admin
,antinuke untrust @user
Remove from whitelist. Identical to ,antinuke unwhitelist @user.
AN Admin

Viewing & Resetting

,antinuke settings
Full overview — status, punishment, log channel, all thresholds, modules, AN admins, and whitelisted users.
AN Admin
,antinuke trusted
List all AN admins and whitelisted users in separate groups.
AN Admin
,antinuke cleanup
Fully reset all antinuke settings to defaults. Clears admins, whitelist, custom thresholds, and modules.
Admin

Features › Last.fm

Last.fm

Show what you're listening to, view music stats, and explore your top artists, tracks and albums

Getting Started

scar integrates with Last.fm to let members share their music. Last.fm tracks every song you listen to on Spotify (or any connected app) as a scrobble. Once you link your account, you can show your now playing, top artists, top tracks, and more.

ℹ️
You'll need a free Last.fm account. Connect it to Spotify at last.fm/settings/applications so your listens get tracked automatically.

Linking your account

,fm set <username>
Link your Last.fm account instantly. You can change it up to 3 times per hour.
Everyone
,fm unlink
Unlink your Last.fm account from this server.
Everyone

Now Playing

The default ,fm command shows what you're currently playing — or the last track you listened to. Displays the track name, artist, album art, and your total scrobble count.

,fm
Show your currently playing or last played track with album art and scrobble count.
Everyone
,fm @user
Check what someone else is listening to. They must have a Last.fm account linked.
Everyone

Stats & History

,fm profile
Full profile overview — total scrobbles, account age, top 3 artists, and now playing. Aliases: ,fm p, ,fm stats
Everyone
,fm recent
Your last 10 scrobbled tracks, with the currently playing track highlighted. Alias: ,fm r
Everyone

Top Charts

All top chart commands support a time period as an optional second argument. If no period is given, it defaults to overall (all time).

💡
Available periods: 7day · 1month · 3month · 6month · 12month · overall
Example: ,fm ta 7day — your top artists from the last 7 days.
,fm topartists [period]
Your top 10 most listened artists with play counts and thumbnail art. Aliases: ,fm ta, ,fm artists
Everyone
,fm toptracks [period]
Your top 10 most played tracks with play counts. Aliases: ,fm tt, ,fm tracks
Everyone
,fm topalbums [period]
Your top 10 most played albums with art. Aliases: ,fm tal, ,fm albums
Everyone
