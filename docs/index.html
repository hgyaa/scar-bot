scar Documentation

A feature-rich Discord bot — moderation, economy, levels, family, Last.fm, giveaways, and much more

Welcome to scar

scar is a powerful all-in-one Discord bot covering every aspect of running a server — anti-nuke protection, deep moderation, a full coin economy, chat and voice levelling, Last.fm music tracking, a family tree system, custom embed builder, giveaways, and dozens more features.

The default command prefix is , (comma). Change it any time with ,prefix [new]. If you forget your prefix, just @mention scar.

ℹ️
All commands use the default , prefix below. Replace it with your server's custom prefix if you've changed it.

Get Started › Quick Start

Quick Start

Get scar fully running in under 5 minutes

1. Add scar to your server

Click Add to Discord in the nav and authorise scar. Grant Administrator for simplest setup, or at minimum: Manage Roles, Ban Members, Kick Members, Manage Channels, Manage Messages.

⚠️
Critical: Drag the scar bot role above any roles you want it to manage. Discord requires the bot's role to sit higher than target roles.

2. Change the prefix (optional)

,prefix !
Change the command prefix. Any character(s) work. If you forget it later, just @mention scar.
Admin

3. Grant staff permissions

,mod let @user
Give a staff member Mod-level access — warn, kick, timeout, mute, lock channels, jail, etc.
Admin
,admin let @user
Give a staff member Admin-level access — ban, clear, full configuration suite.
Admin

4. Run essential setup commands

,welcome set #channel Welcome {user.mention}!
Set your welcome message. Supports embed scripts and variables.
Admin
,log set #mod-logs
Enable full server logging to a dedicated channel.
Admin
,setup jail
Auto-create the jail role and channel.
Admin
,setup mute
Auto-create the mute role and configure all channel permissions.
Admin
,antinuke toggle on
Enable AntiNuke protection. Nothing is active until you run this.
Admin

5. Explore with ,help

Run ,help in Discord for an interactive overview. Use ,help moderation, ,help economy, ,help fun, etc. for category-specific command lists.

Moderation › AntiNuke

AntiNuke

Real-time server protection against mass destructive actions

How It Works

AntiNuke watches your audit log in real time. When someone triggers a destructive action above a threshold within a 10-second window, scar automatically punishes them before they can do further damage.

🛡️
Enable first: Nothing is active until you run ,antinuke toggle on.

Enable & Configure

,antinuke toggle on/off
Enable or disable the entire AntiNuke system. Also ,an toggle on/off.
Admin
,antinuke threshold [action] [number]
Set how many actions trigger punishment within 10 seconds. e.g. ,an threshold ban 3.
Admin
,antinuke punishment [ban/kick/strip]
Set the punishment: ban, kick, or strip (remove all roles).
Admin
,antinuke all on/off
Toggle all protection modules at once.
Admin
,antinuke modules
View all protection modules and their current on/off status.
Admin

Protected Events

  • bans — mass member banning
  • kicks — mass member kicking
  • channel_delete — bulk channel deletion
  • role_delete — bulk role deletion
  • webhook_create — suspicious webhook creation
  • bot_add — unauthorized bot additions
  • role_update — mass permission escalation

Trust & Whitelist

,antinuke admin @user
Add a trusted admin exempt from AntiNuke triggers. Use ,an unadmin @user to remove.
Admin
,antinuke trust @user
Whitelist a user so their actions never trigger AntiNuke. Also ,an whitelist @user.
Admin
,antinuke untrust @user
Remove a user from the trust/whitelist.
Admin
,antinuke log #channel
Set a channel to receive AntiNuke action logs.
Admin
,antinuke list
View all thresholds, current punishment, and trusted users.
Admin
,antinuke settings
Full config overview — all modules, thresholds, punishment, and trusted list.
Admin
